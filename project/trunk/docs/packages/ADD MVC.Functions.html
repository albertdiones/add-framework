    handle_error() callback function for handling trigger_error() errors

    handle_error(int $errno, string $errstr, string $errfile, int $errline, array $errcontext) 

    see \http://www.php.net/manual/en/function.set-error-handler.php
    version 1.0

    Parameters

    $errno

    int

    contains the level of the error raised, as an integer.

    $errstr

    string

    contains the error message, as a string.

    $errfile

    string

    contains the filename that the error was raised in, as a string.

    $errline

    int

    contains the line number the error was raised at, as an integer.

    $errcontext

    array

    an array that points to the active symbol table at the point the error occurred. In other words, errcontext will contain an array of every variable that existed in the scope the error was triggered in. User error handler must not modify error context.
